# Godbound: Paradoxical Archive Options

A Foundry VTT module for the Godbound system that adds optional rules inspired by the Paradoxical Archive.

## Features
- **Paradoxical Archive HP**: Checkbox to double Constitution score as base HP + (half Con score + Con mod) per level after 1.
- **Paradoxical Archive Damage**: Checkbox placeholder (damage modification not yet fully implemented).
- **Starting Effort Override**: Number field to set custom Effort at level 1.

## Installation
In Foundry VTT:
1. Go to **Add-on Modules** → **Install Module**.
2. Paste the manifest URL:  
   `https://raw.githubusercontent.com/YOUR-USERNAME/godbound-pa-options/main/module.json`
3. Install and enable.
